Документация Multiuser Dungeon Game
==================================================
MOOD - это многопользовательский подземелье, где для отображения монстров применяется python-cowsay. Игроки начинают игру на игровом поле размером 10x10 клеток с координатами (0, 0) и могут передвигаться в любом направлении, сталкиваться с монстрами, атаковать их разнообразным вооружением, а также взаимодействовать с другими участниками.
